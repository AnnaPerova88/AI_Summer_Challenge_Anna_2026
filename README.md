**день 2 из 92 (02.06.2026)**

**AI English Conversation Tutor 🎓**
Интерактивный AI-репетитор английского языка с живым диалогом, проверкой грамматики и рекомендацией новых слов. Работает на модели Qwen2.5-1.5B-Instruct с оптимизацией для мощных GPU.

Возможности
💬 Живой диалог — AI задаёт вопросы и поддерживает беседу
📝 Мягкая коррекция — исправление грамматических ошибок с объяснениями на русском
📚 Новая лексика — рекомендация слов по теме разговора
🎯 12 тем и 5 уровней (от A1 до C1)
⚡ Мгновенные ответы — оптимизация под GPU (bfloat16)
Технологии
Python, PyTorch, Transformers (Hugging Face)

Модель: Qwen/Qwen2.5-1.5B-Instruct

Gradio UI — веб-интерфейс

**день 3 из 93 (03.06.2026)**   

**DataInterview Prep 🚀**
Платформа для подготовки к собеседованиям по Data Engineering, ML и Data Science. Включает AI-оценку ответов, поиск вакансий и аналитику рынка.

Возможности
🎯 Практика интервью — база 400+ вопросов по SQL, PySpark, Airflow, ML, MLOps
🤖 AI-наставник — проверка ответов с обратной связью
💼 Вакансии с hh.ru — подбор по направлениям
📈 Аналитика навыков — востребованность технологий на рынке
📚 Roadmap — структурированный план обучения

Технологии
Python, Gradio, Pandas, Requests

API hh.ru
AI-оценка (в т.ч. через локальные LLM)
